## 用法

Windows 10以上系统运行`install.bat`，快速完成资源包的导入

Android系统按照`zip_exclude.txt`中的排除条件压缩文件`*.mcaddon`，并用Minecraft打开此文件

`export_mcworld.bat`用于快速导出世界存档（供开发测试用）
